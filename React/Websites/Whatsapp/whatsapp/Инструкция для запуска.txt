Для локального запуска проекта необходимо сохранить файлы на локальную машину и в командной строке прописать команду:

npm start

После этого откроется окно в браузере и необходимо будет пройти этап авторизации пользователя введя, соответственно, пользовательский "IdInstance" и "ApiTokenInstance".

После этого можно начать пользоваться данным приложением. 

Изначально приведены 3 чата, которые показывают внешний вид зарегестрированных пользователей.

Для создания нового чата необходимо нажать на иконку сообщения, которая находится под id пользователя в графе слева. После этого необходимо ввести мобильный телефон и имя пользователя.

Далее можно перейти в чат с данным пользователем.

Для того, чтобы выйти из аккауна необходимо нажать на иконку выхода, которая находится справа от иконки создания нового чата.
